# Responsive Single Page Website

A modern and responsive single-page website built with HTML and CSS. This project was created as a practice exercise to improve my front-end development skills and understanding of UI design.

---

## 📖 Overview

This project is a single-page website with a clean layout and smooth navigation between sections. It includes two versions:

- ☀️ Light Mode
- 🌙 Dark Mode

Both versions are included in this repository to practice theme switching and UI styling differences.

---

## ✨ Features

- Fully responsive design
- RTL (Right-to-Left) layout support
- Sticky navigation bar
- Smooth scrolling between sections
- Hero (header) section
- About section
- Services section
- Contact form
- Footer section
- Light & Dark mode versions

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
